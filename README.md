# Skills Pack

本仓库包含所有可用的AI技能集合。

## 技能列表

| Skill名称 | 描述 | 详情链接 |
| --- | --- | --- |
| memory-manager | 跨会话分层记忆管理skill，帮助AI助手在会话结束时提炼关键信息写入结构化记忆文件，在下次会话开始时读取并恢复上下文，实现持久化记忆。适用于任何领域（编程、写作、研究、运营等）和任何支持文件读写的AI工具。 | [memory-manager.md](./memory-manager.md) |
| round-table | 通用的多Agent圆桌决策系统，通过独立subagent实现不同角色人格，从产品、技术、风险等多个角度讨论决策，避免思维盲区。 | [roundtable/roundtable.md](./roundtable/roundtable.md) |
